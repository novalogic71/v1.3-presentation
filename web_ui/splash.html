Key Features

Sub-Frame Accuracy

98% accuracy improvement with frame-perfect timing. Displays exact offsets like "-240f @ 23.976fps" for precise sync correction.

Multi-GPU Acceleration

Automatic workload distribution across all available GPUs with round-robin load balancing. Process 15-20 files per hour.

AI-Powered Detection

Wav2Vec2, YAMNet, and custom models with 768-1024 dimensional embeddings for complex sync scenarios.

Frame Rate Detection

Uses ffprobe to detect actual video frame rates (23.976, 24, 29.97 fps, etc.) for accurate frame count displays.

Batch Processing

CSV-based workflows with 1-8 parallel workers. Process up to 100 file pairs with real-time progress monitoring.

Automated Repair

One-click sync correction with A/B preview comparison before applying repairs. Includes comprehensive packaging.
